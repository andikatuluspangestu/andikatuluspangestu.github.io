B. Manfaat menggunakan Whatsapp Mod

  • Bisa menikmati Fitur Whatsapp terkini Kebanyakan Whatsapp Mod menggunakan Versi BETA terkini, sehingga banyak Fitur baru yang tidak didapatkan di Whatsapp Stabil yang ada di Playstore.
  • Fitur Privasi lebih lengkap Fitur Privasi merupakan fitur utama yang di bawa oleh pengembang Whatsapp Mod dan menjadi daya tarik tersendiri.
  • Resolusi Media Dengan menggunakan Whatsapp Mod, kita bisa mengirim dan menerima, bahkan mengunggah story whatsapp dengan kualitas yang lebih baik daripada Whatsapp original.

C. Fitur Privasi Whatsapp Mods

  • Menyembunyikan Centang Biru
  • Menyembunyikan Centang Dua

  • Melihat Pesan yang dihapus

F. Backup Pesan ( Langkah-langkah mencadangkan Pesan Whatsapp )

Sebelum beralih menggunakan Whatsapp Modifikasi, alangkah baiknya kita mencadangkan semua pesan ke cadangan lokal, sehingga bisa kita pulihkan di Whatsapp mod nantinya.

Backup Whatsapp di Android 4.1 - Android 10 Q :

  1. Buka Aplikasi Whatsapp
  2. Tekan “Titik Tiga” di Pojok Kanan Atas
  3. Pilih dan Buka Setelan
  4. Masuk ke bagian “Chat”
  5. Geser kebawah, dan tekan menu “Cadangan Chat”
  6. Tekan Tombol “CADANGKAN” dan Tunggu sampai selesai ( 100% )
  7. Setelah selesai, pastikan terdapat Informasi Waktu Terakhir Pencadangan.

  1. Buka di File manager
  2. Masuk ke Direktori Internal -> WhatsApp -> Databases

  1. Lihat apakah terdapat File Database atau tidak, jika ada maka lanjutkan Langkah berikutnya dan jika tidak ada maka lakukan ulang pencadangan-nya.

Backup Whatsapp di Android 11 - Android 12 :

  1. Buka Aplikasi Whatsapp
  2. Tekan “Titik Tiga” di Pojok Kanan Atas
  3. Pilih dan Buka Setelan
  4. Masuk ke bagian “Chat”
  5. Geser kebawah, dan tekan menu “Cadangan Chat”
  6. Tekan Tombol “CADANGKAN” dan Tunggu sampai selesai ( 100% )
  7. Setelah selesai, pastikan terdapat Informasi Waktu Terakhir Pencadangan.

  1. Buka di File manager
  2. Masuk ke Direktori Internal -> Android -> Media -> com.whatsApp -> WhatsApp -> Databases

  1. Lihat apakah terdapat File Database atau tidak, jika ada maka lanjutkan Langkah ke-11 dan jika tidak ada maka lakukan ulang pencadangan-nya.
  2. Ubah nama folder com.whatsapp menjadi com.whatsapp4

H. Penjelasan Menu Privasi Whatsapp Mod

Untuk memodifikasi atau menghidupkan konfigurasi privasi di Whatsapp Mod, kamu hanya perlu ketuk “Titik Tiga” di Pojok Kanan atas, lalu buka menu “Privasi”

  • Hide / Freeze Last Seen ( Bekukan Terakhir dilihat ) Membekukan Status Terakhir dilihat
  • Anti Revoke ( Anti Hapus Pesan ) Pesan terhapus, masih bisa dilihat
  • Blue Tick after Reply ( Sembunyikan Centang Biru hanya setelah balas ) Perlihatkan Centang Biru ke kontak hanya ketika kamu telah membalas pesannya
  • Hide Status View ( Sembunyikan Melihat Status ) Kontak tidak akan tahu, bahwa kamu telah melihat status dia
  • Hide Blue Tick ( Sembunyikan Centang Biru ) Menyembunyikan Centang Biru dari kontak, walaupun kamu telah membaca pesannya
  • Hide Second Tick ( Sembunyikan Centang Dua ) Menyembunyikan Centang Dua dari kontak, walaupun kamu telah menerima atau juga membaca pesannya
  • Always Online ( Selalu Online ) Mengaktifkaan Status Terakhir dilihat menjadi selalu Online
  • Hide Forwarder Tag Menyembunyikan Simbol Pesan diteruskan
  • Anti-View Once Walaupun Gambar/Foto di setel hanya sekali lihat, namun kamu bisa selalu melihatnya
  • Anti-Delete Status Kamu akan selalu bisa melihat status kontak yang telah dihapus
  • Hide Blue Microphone ( Sembunyikan Microphone Biru ) Sembunyikan Simbol Microphone Biru, walau kamu telah mendengarkan voice note dari kontak kamu
  • Hide Recording ( Sembunyikan Sedang Merekam ) Sembunyikan notifikasi dan tidak memberi tahu kontak bahwa kamu Sedang Merekam
  • Hide Typing ( Sembunyikan Sedang Mengetik ) Sembunyikan notifikasi dan tidak memberi tahu kontak bahwa kamu Sedang Mengetik

I. Penjelasan Menu Lanjutan/Advance Whatsapp Mod

Untuk memodifikasi atau menghidupkan konfigurasi lainnya di Whatsapp Mod, kamu hanya perlu ketuk “Titik Tiga” di Pojok Kanan atas, lalu buka gulir ke bawah dan pilih menu “Pengaturan”, lalu pilih “Pengaturan Lebih Lanjut”

Btw Untuk mengakses Fitur ini direkomendasikan untuk menggunakan Whatsapp Go : Unduh Sekarang

  • Berbagi Media Kamu bisa memodifikasi konfigurasi tentang Media, seperti Kualitas Gambar, Video dan sebagainya.

  • Cadangkan & Pulihkan Fitur ini tidak terlalu berguna menurut saya pribadi, dan saya tidak pernah menggunakannya.

  • Bersihkan File Whatsapp Kamu bisa membersihkan Cache, Logs, dan Cadangan Obrolan lama. Sehingga bisa melegakan kapasitas memori internal HP kamu.

  • Pengaturan Lainnya Di dalam menu ini, kamu akan melihat beberapa konfigurasi seperti : a ) Sembunyikan Tombol Unduh : Menyembunyikan Tombol Unduh Media Status b ) Konfirmasi Pengiriman Sticker : Menampilkan POP-UP Konfirmasi, sebelum sticker di kirim c ) Ikon DND : Jika diaktifkan maka tombol Mode Pesawat atau Mode Offline akan di sembunyikan dari halaman depan

    Beberapa fitur atau menu mungkin tidak tersedia seperti diatas,
karena mungkin Developer telah meperbaruinya atau menghilangkannya.
